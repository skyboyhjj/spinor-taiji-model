四、表格设计规范

📐 尺寸规范：
- 最大宽度：677px（微信公众号编辑器容器宽度）
- 字体大小：14px（表格内容），12px（紧凑表格）
- 单元格内边距：10px 8px
- 行高：自然换行，使用 word-break: break-all
🎨 样式规范：
- 表头：深红色渐变背景（rgba(139,30,34,0.1) → rgba(139,30,34,0.05)）
- 表体：隔行变色（浅灰色 rgba(0,0,0,0.03)）
- 边框：底部细线分隔（1px solid #eee）
- 文字颜色：表头#333，表体#555
✅ 测试验证步骤：
1. 在微信公众号编辑器中粘贴表格HTML
2. 预览手机端效果，检查是否需要横向滚动
3. 在平板（iPad）上预览，检查布局是否正常
4. 在桌面浏览器中预览，检查居中对齐
5. 检查长文本是否自动换行，无内容被截断

五、实现代码示例

以下是生成上述表格的Python代码示例：

from WeChatTableGenerator import WeChatTableGenerator

generator = WeChatTableGenerator()

# 标准表格
headers = ['境界', '禅宗', '认知状态']
rows = [['第一层', '看山是山', '完全认同现象'], ...]
table_html = generator.generate_table(headers, rows, caption='表格标题')

# 多列表格（自动转换为卡片布局）
table_html = generator.generate_card_layout(headers, rows)

# 键值对汇总表
summary_data = {'键1': '值1', '键2': '值2'}
table_html = generator.generate_summary_table(summary_data, title='汇总标题')
