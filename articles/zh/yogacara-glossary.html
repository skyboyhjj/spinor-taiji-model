唯识学词汇表

以下定义，基于《唯识三十颂》（玄奘译）原文、传济法师讲解系列及旋量-太极模型知识库交叉整理。
名可名，非常名——这些定义是可操作的比喻脚手架，非最终真理。

一、唯识框架

三能变 唯识框架

《三十颂》总纲。初能变（第八阿赖耶识·异熟能变）→ 二能变（第七末那识·思量能变）→ 三能变（前六识·了别境能变）。"此能变为三，谓异熟思量，及了别境识。"

出处：第1-2颂

相关词条：阿赖耶识末那识前五识第六意识
一切唯识 唯识框架

唯识学根本论题。"由假说我法，有种种相转，彼依识所变"——一切我法皆是识所变现的假说。"是诸识转变，分别所分别，由此彼皆无，故一切唯识"——无客观外境。

出处：第1颂·第17颂

相关词条：假说我法分别·所分别
假说我法 唯识框架

佛随顺众生妄执，方便建立"我"（主宰）"法"（轨持）二相。分二种假——无体随情假（凡夫妄执安立）与有体施设假（佛为开示实相方便施设）。"由假说我法，有种种相转。"

出处：第1颂

相关词条：一切唯识四分说
三时教判 唯识框架

有教（阿含·说有）→ 空教（般若·说空）→ 中道教（唯识·说中道）。唯识为破"执有""执空"两边而立中道义。

出处：传济法师讲解01（依《解深密经》三时判教）

相关词条：一切唯识

二、八识体系

阿赖耶识 八识体系

第八识，初能变，异熟一切种。三名——阿赖耶（自相·能藏/所藏/执藏）、异熟识（果相）、一切种识（因相）。无覆无记，恒转如瀑流，阿罗汉位舍染污名不捨体。

出处：第2-4颂

相关词条：种子恒转如瀑流无覆/有覆无记大圆镜智
末那识 八识体系

第七识，二能变。恒审思量执第八见分为我。"依彼转缘彼，思量为性相。"——依第八识种子及现行生起，又向内攀缘第八见分。有覆无记。

出处：第5-7颂

相关词条：四根本烦恼有覆无记平等性智
前五识 八识体系

眼耳鼻舌身五识，了别境能变。"如波涛依水"——依根本识（第八阿赖耶识）随缘现起。唯现量取境，无深分别。

出处：第8颂·第15颂

相关词条：第六意识五位无心
第六意识 八识体系

了别境能变主力。通善·不善·无记三性，通过去·现在·未来三时，通三界九地。造业力最强。"意识常现起，除生无想天，及无心二定，睡眠与闷绝。"

出处：第8颂·第16颂

相关词条：前五识五位无心心所体系
四根本烦恼 八识体系

末那识恒与四烦恼相应：我痴（无明·迷无我理）→ 我见（妄计为我）→ 我慢（倨傲高举）→ 我爱（贪着所执我）。"四烦恼常俱。"俱生我执的四根支柱。

出处：第6颂

相关词条：末那识遍计所执性
无覆/有覆无记 八识体系

第八阿赖耶识 = 无覆无记（不覆障本心）；第七末那识 = 有覆无记（覆障圣道，但非善非恶不造业）。三性归属是理解八识功能差异的关键。

出处：第4颂·第7颂

相关词条：阿赖耶识末那识
五位无心 八识体系

意识不现行的五种状态——无想天（生无想报）·无想定（凡夫外道）·灭尽定（三乘圣者）·极重睡眠·闷绝。"意识常现起，除此五例外。"

出处：第16颂

相关词条：第六意识
恒转如瀑流 八识体系

第八识种子与现行刹那生灭、相似相续，如瀑布奔流不息。"恒转如瀑流，阿罗汉位舍"——阿罗汉位舍"阿赖耶识"染污名，本体仍存转名异熟识。

出处：第4颂

相关词条：阿赖耶识种子
四分说 八识体系

相分（所缘影像）→ 见分（能缘作用）→ 自证分（自证见分缘相分）→ 证自证分（证自证分缘自证分）。护法菩萨四分说为玄奘传承正脉。传济法师蜗牛喻：头=自证分，两角=见分与相分。

出处：《成唯识论》护法菩萨（传济法师讲解04）

相关词条：能所分别·所分别
能所 八识体系

能证与所证的三层深度——表层（见分-相分）→ 中层（自证分-见分）→ 深层（证自证分-自证分）。真正的能所双亡需消融四分中所有对待，即圆成实性。

出处：唯识学四分说（知识库文档）

相关词条：四分说圆成实性
分别·所分别 八识体系

能取（见分·我执）与所取（相分·法执）的二元结构。"是诸识转变，分别所分别"——一切认知皆是识自身的二分显现，非外在客观存在。

出处：第17颂

相关词条：四分说二取随眠
心所体系 八识体系

第9-14颂详列51心所——遍行5·别境5·善11·烦恼6·随烦恼20·不定4。心的功能而非造作动作，五遍行实际刹那同时。

出处：第9-14颂

相关词条：五遍行心所别境心所
五遍行心所 八识体系

触（根境识和合）·作意（警觉引心）·受（领纳起爱）·想（取相安名）·思（令心造作）。遍一切心、一切地、一切时、一切性的五种基本心理功能。阿赖耶识唯与五遍行相应。

出处：第3颂·第10颂

相关词条：阿赖耶识别境心所
别境心所 八识体系

欲·胜解·念·定·慧。缘各别境方得生起的五种心所。"所缘事不同"——各自所缘境界对象不同。修定的心所基础。

出处：第10颂

相关词条：五遍行心所

三、种子理论

种子 种子理论

阿赖耶识中能生起特定结果的功能潜能，不是实体而是功能流。分本有种子与新熏种子、有漏种子与无漏种子。"一切种识"含藏宇宙一切可能。

出处：第2颂·第18颂

相关词条：阿赖耶识现行熏习种子六义
种子六义 种子理论

刹那灭（生已即灭）·果俱有（因果同时）·恒随转（持续流转）·性决定（善种生善果）·待众缘（须众缘和合）·引自果（各引自类果）。定义种子资格的六项法定条件。

出处：《成唯识论》卷二（唯识学核心概念）

相关词条：种子性决定
性决定 种子理论

"莫之命而常自然"——善种子只能生善果，恶种子只能生恶果。功能即性质，不是道德法则而是物理定律。

出处：唯识学性决定（知识库文档）

相关词条：种子种子六义
有漏种子 种子理论

被烦恼所染的种子。"不能被消灭，只能不退"——超过而非消灭。唯识学工程智慧的核心难题：不正面压制而是改变整体生态。

出处：唯识学深度读解（知识库文档）

相关词条：无漏种子熏习
无漏种子 种子理论

未被烦恼所染的清净种子。愿力所熏，转依之资。"修行不是除草，是多种花。"

出处：唯识学进阶（知识库文档）

相关词条：有漏种子转识成智
现行 种子理论

种子遇缘成熟时所生起的具体精神或物质活动。"以展转力故，彼彼分别生"——种子生现行。

出处：第18颂

相关词条：种子熏习
熏习 种子理论

现行反熏阿赖耶识留下新种子的过程。经验自动沉积，如茶壶留茶气、牛奶加奶粉变浓。种子↔现行双向循环是生命持续演化的根本机制。

出处：传济法师讲解04

相关词条：种子现行阿赖耶识
业习气 种子理论

由善恶业力熏成的潜在势力。"由诸业习气，二取习气俱，前异熟既尽，复生余异熟"——异熟轮回的直接驱动力。

出处：第19颂

相关词条：种子异熟

四、三性三无性

遍计所执性 三性三无性

"由彼彼遍计，遍计种种物，此遍计所执，自性无所有"——虚妄分别，把被平方出来的矢量执着为实有。一切执着与无明的根源。绳蛇喻：夜见绳为蛇，蛇=遍计执。

出处：第20颂

相关词条：依他起性圆成实性末那识
依他起性 三性三无性

"依他起自性，分别缘所生"——众缘（亲因缘·所缘缘·等无间缘·增上缘）和合而生。涵盖一切有为法。绳蛇喻：绳（麻编而成）=依他起。"故此与依他，非异非不异"（与圆成实一体两面）。

出处：第21-22颂

相关词条：遍计所执性圆成实性
圆成实性 三性三无性

"圆成实于彼，常远离前性"——于依他起上远离遍计所执所显之真如。绳蛇麻喻：麻（绳之体）=圆成实。不生不灭，不垢不净，与中观中道指向同一终极实在。

出处：第21-22颂

相关词条：真如依他起性
三无性 三性三无性

"即依此三性，立彼三无性，故佛密意说，一切法无性"——相无性（依遍计执立·体相毕竟非如空花）→ 无自然性（依依他起立·因缘和合无自然性）→ 胜义无性（依圆成实立·远离前所执我法性）。如筏喻，到岸须舍。

出处：第23-24颂

相关词条：遍计所执性依他起性圆成实性

五、修行次第

唯识五位 修行次第

从初发心至成佛的完整修行次第——资粮位 → 加行位 → 通达位 → 修习位 → 究竟位。前24颂明"唯识相"，第25颂明"唯识性"，第26-30颂明"唯识行位"。

出处：第26-30颂

相关词条：资粮位加行位通达位修习位究竟位
资粮位 修行次第

"乃至未起识，求住唯识性，于二取随眠，犹未能伏灭"——地前三贤（十住·十行·十回向）。修福德智慧二资粮。"修慧不修福，罗汉托空钵；修福不修慧，大象挂璎珞。"凡夫至见道前的漫长准备。

出处：第26颂

相关词条：加行位二取随眠
加行位 修行次第

"现前立少物，谓是唯识性，以有所得故，非实住唯识"——暖·顶·忍·世第一法四位。依明得定、明增定、印顺定、无间定。带相观空，虽证二取空，仍"携真如之相"而观，未亲证真如。

出处：第27颂

相关词条：资粮位通达位
通达位 修行次第

"若时于所缘，智都无所得，尔时住唯识，离二取相故"——见道位 = 初地菩萨（极喜地）。无分别智亲证真如，能所双亡。凡圣分水岭。

出处：第28颂

相关词条：真如加行位
修习位 修行次第

"无得不思议，是出世间智，舍二粗重故，便证得转依"——初地至十地，地地修十波罗蜜、断二障种子习气、证对应真如。（十地展开见传济法师讲解23）

出处：第29颂

相关词条：通达位究竟位转依
究竟位 修行次第

"此即无漏界，不思议善常，安乐解脱身，大牟尼名法"——佛果。大菩提（智）·大涅槃（理）= 二转依果。法身三类：自性身、受用身、变化身。

出处：第30颂

相关词条：无漏界法身三身
二取随眠 修行次第

能取（见分·我执）与所取（相分·法执）的烦恼习气种子，如眠伏藏。资粮位"于二取随眠，犹未能伏灭"——未能伏灭，遇缘即起。

出处：第26颂

相关词条：资粮位分别·所分别二障伏断
无所得 修行次第

"智都无所得"——连"唯识性"也不能执取。有所得故非实住唯识。修行关键转折：从"求"到"无求"。

出处：第27-28颂

相关词条：通达位加行位
转依 修行次第

迷悟依（开灯·转迷为悟）+ 染净依（清理·转染成净）。"舍二粗重故，便证得转依"。"但转其名，不转其体"——转舍依他起上遍计所执，转得依他起中圆成实性。

出处：第29颂

相关词条：修习位转识成智
消业 修行次第

消业 = 不浇水。忏悔·持戒·修善·转依四种方式。不是删除业力种子，而是剥夺其四缘（亲因缘·所缘缘·等无间缘·增上缘）令不退。

出处：唯识学进阶（知识库文档）

相关词条：有漏种子
功德/福德 修行次第

功德 = 无漏善种子势力（三轮体空·不执施者受者施事）；福德 = 有漏善业果报。"修慧不修福，罗汉托空钵。"

出处：唯识学进阶（知识库文档）

相关词条：三轮体空

六、果证·四智

真如 果证·四智

"此诸法胜义，亦即是真如，常如其性故，即唯识实性"——圆成实性 = 唯识实性 = 诸法胜义。不生不灭，不垢不净。前24颂明唯识相，此颂入唯识性。

出处：第25颂

相关词条：圆成实性通达位
转识成智 果证·四智

将有漏八识转化为无漏四智——前五识 → 成所作智（眼·办事的手）、第六意识 → 妙观察智（观察的眼）、末那识 → 平等性智（平等的心）、阿赖耶识 → 大圆镜智（如如的体）。"但转其名，不转其体。"

出处：《成唯识论》卷十（传济法师讲解08·23）

相关词条：大圆镜智平等性智
大圆镜智 果证·四智

第八阿赖耶识彻底转依后的状态。如大圆镜遍照一切而不沾不染。四智之体——体（大圆镜）·心（平等性）·眼（妙观察）·手（成所作）一体关系。与真如一体两面，如手翻覆。

出处：传济法师讲解08

相关词条：阿赖耶识真如
平等性智 果证·四智

第七末那识转依后。"水泡回归大海"——破我执后所见皆影像。初地下品（转分别我执）→ 八地中品（任运破俱生）→ 佛果上品（究竟转）。

出处：传济法师讲解09

相关词条：末那识大圆镜智
出世间智 果证·四智

"无得不思议，是出世间智"——超越主客二元、言语道断的觉悟智慧。见道位（通达位）初次生起。

出处：第29颂

相关词条：通达位无所得
法身三身 果证·四智

"大牟尼名法"——自性身（真净法界·一切法平等实性）、受用身（自受用·他受用为十地菩萨说法）、变化身（随类化身·居净秽土度众）。放大镜喻：法界无中心，何处念何处见。

出处：第30颂

相关词条：究竟位无漏界
无漏界 果证·四智

"此即无漏界"——真无漏非相对有无，如气球大到无边际一体故无可漏。"佛界即众生界，娑婆即极乐"——非二处。

出处：第30颂

相关词条：究竟位
二障伏断 果证·四智

烦恼障（我执）与所知障（法执）的伏断次第——资粮位未伏 → 加行位伏分别二取现行 → 通达位断分别二障种子 → 修习位地地断俱生二障 → 究竟位永尽。

出处：第26-30颂

相关词条：二取随眠唯识五位
三轮体空 果证·四智

不执著施者（我）·受者（你）·施事（所行之事）。行动之后即刻将执着消解。"善行无辙迹"——从源头平方不留痕迹。

出处：《金刚经》唯识学进阶（知识库文档）

相关词条：功德/福德

七、法师讲释特色

次第学法 法师讲释

传济法师反复强调的学修路径——《大乘百法明门论》→《八识规矩颂》→《唯识二十论》→《唯识三十颂》→《成唯识论》。"名相不重复释靠次第承递"，先打好百法基础再入三十颂。

出处：传济法师讲解04·08

相关词条：三能变
生活比喻 法师讲释

传济法师以丰富比喻使唯识通俗化——蜗牛喻四分（头=自证分·两角=见相二分）、绳蛇麻喻三性、茶壶留茶气喻熏习、程序代码喻心识变现、相机成像喻见相二分、气球喻无漏界。

出处：传济法师讲解04-24

相关词条：四分说三自性
科学类比 法师讲释

传济法师善用现代科学类比——基因=共业种子（现代遗传学）、视频帧喻刹那生灭、观测者效应近"识变"、AI大数据喻佛任运度生。

出处：传济法师讲解07·11·24
