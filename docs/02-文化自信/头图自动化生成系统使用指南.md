### 🎯 核心功能

| 功能 | 说明 |
|------|------|
| ✅ 自动触发 | 文章制作完成后自动生成头图 |
| ✅ 智能命名 | "文章标题+日期"格式命名 |
| ✅ 分类保存 | 按文章分类保存到指定文件夹 |

## 三、快速开始

### 3.1 基本使用

```python
from auto_cover_generator import ArticleCoverWorkflow

# 创建工作流
workflow = ArticleCoverWorkflow()

# 为文章生成头图
result = workflow.create_article_cover(
    article_title="看山三境界",
    article_dir="Spinor-Taiji model",
    style="three_realms"
)

# 检查结果
if result.success:
    print(f"✅ 头图已生成：{result.path}")
else:
    print(f"❌ 生成失败：{result.error}")
```

### 3.2 批量生成

```python
from auto_cover_generator import ArticleCoverWorkflow

workflow = ArticleCoverWorkflow()

# 定义批量任务
tasks = [
    {
        'article_title': '旋量-太极',
        'article_dir': 'Spinor-Taiji model',
        'prompt': '你的自定义提示词',
        'size': '16:9'
    },
    {
        'article_title': '看山三境界',
        'article_dir': 'Spinor-Taiji model',
        'prompt': '你的自定义提示词',
        'size': '16:9'
    }
]

# 批量生成
results = workflow.generator.batch_generate(tasks)
```

### 3.3 专用生成器使用

```python
# 使用看山三境界专用生成器
from generate_three_realms_cover import generate_three_realms_cover

result = generate_three_realms_cover()

if result['success']:
    print(f"✅ 头图路径：{result['path']}")
```

## 四、配置说明

### 4.1 配置文件

编辑 `cover_config.json` 来自定义系统行为：

```json
{
  "generation_settings": {
    "default_size": "16:9",        // 默认尺寸
    "default_quality": "high",     // 默认质量
    "max_retries": 3,              // 最大重试次数
    "retry_delay": 5,              // 重试间隔（秒）
    "image_format": "JPG"          // 图像格式
  },
  
  "dimensions": {
    "16:9": {
      "width": 1200,
      "height": 630,
      "dpi": 300
    }
  }
}
```

### 4.2 尺寸规格

| 规格 | 宽×高 | DPI | 用途 |
|------|-------|-----|------|
| 16:9 | 1200×630 | 300 | 微信公众号封面（推荐） |
| 1:1 | 1080×1080 | 300 | 正方形封面 |
| 4:3 | 1200×900 | 300 | 横版大图 |

## 七、错误处理

### 7.1 错误类型

| 错误类型 | 代码 | 说明 | 处理方式 |
|---------|------|------|---------|
| 网络错误 | E001 | API调用失败 | 自动重试3次 |
| 超时错误 | E002 | 请求超时 | 自动重试 |
| 文件错误 | E003 | 保存失败 | 检查路径权限 |
| 参数错误 | E004 | 参数无效 | 返回明确错误信息 |

### 7.2 重试机制

```python
# 系统自动重试配置
max_retries = 3          # 最大重试次数
retry_delay = 5          # 重试间隔（秒）
exponential_backoff = True  # 指数退避

# 示例：首次失败5秒后重试，第二次10秒后重试
```

### 7.3 错误提示示例

```python
if not result.success:
    print(f"❌ 头图生成失败")
    print(f"💬 错误代码：{result.error_code}")
    print(f"💬 错误信息：{result.error_message}")
    print(f"🔄 建议：请检查网络连接后重试")
```

## 十一、故障排查

### 问题1：生成失败

**症状**：返回 `success=False`

**解决方案**：
1. 检查网络连接
2. 验证豆包API配置
3. 查看错误日志
4. 重试生成

### 问题2：图像不清晰

**症状**：图像模糊或失真

**解决方案**：
1. 检查DPI设置（应为300）
2. 验证尺寸比例
3. 使用高质量模式

### 问题3：路径错误

**症状**：文件保存失败

**解决方案**：
1. 检查目录是否存在
2. 验证写入权限
3. 清理文件名非法字符
