# 📤 图片手动上传指南

## 一、准备工作

### 1. 图片规格要求
- 格式: JPEG (.jpg)
- 尺寸: 1080 × 608 像素 (16:9 横版)
- 文件大小: 建议 < 500KB
- 命名格式: 描述性名称.jpg

### 2. 图片目录
```
02-文化自信/images/
├── spinor.jpg          # 旋量示意图
├── taiji.jpg           # 太极图
└── emptiness.jpg       # 从空到有示意图
```

## 二、上传步骤

### 步骤1: 准备图片文件
确保图片符合以下要求:
- ✅ 文件格式为 JPEG
- ✅ 尺寸为 1080 × 608 像素
- ✅ 文件名为英文或中文（无特殊字符）

### 步骤2: 复制到图片目录
将图片复制到:
```
02-文化自信/images/
```

### 步骤3: 更新HTML文件
运行以下命令更新HTML引用:
```bash
python update_image_paths.py
```

### 步骤4: 验证
```bash
python validate_images.py
```

## 三、验证检查清单

| 检查项 | 要求 | 状态 |
|-------|------|------|
| 文件存在 | 3张图片都在images目录 | ✅ |
| 文件格式 | JPEG (.jpg) | ✅ |
| 文件大小 | < 500KB | ✅ |
| 图片尺寸 | 1080 × 608 | ✅ |
| HTML引用 | 正确指向图片 | ✅ |

## 四、工具命令

```bash
# 更新图片路径
python update_image_paths.py

# 验证图片配置
python validate_images.py

# 启动测试服务器
python start_server.py
```

## 五、故障排除

### 问题1: 图片不显示
- 检查文件路径是否正确
- 确保图片文件存在
- 验证文件格式为JPEG

### 问题2: 图片显示破损
- 检查图片文件是否损坏
- 重新上传图片
- 验证文件大小是否正常

### 问题3: HTML引用错误
- 运行 update_image_paths.py 更新引用
- 检查文件名是否正确
