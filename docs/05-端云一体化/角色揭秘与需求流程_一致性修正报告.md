## 二、各点建议实施详情

### 2.1 建议一：增加AI慧惠角色章节

**实施内容**：

在《慧惠项目角色揭秘.md》中新增"角色零：AI慧惠——北极星定位的陪伴者"章节，包含以下内容：

| 小节 | 内容 | 说明 |
|------|------|------|
| **核心定位** | "譬如北辰，居其所而众星共之" | 北极星定位，不替用户选择路径 |
| **行事准则** | 做减法、敢于遗忘、不堆砌知识、不告诉"你应该" | 明确慧惠的行事原则 |
| **核心职责** | 需求初步筛选、内容生成辅助、反馈收集、道境四维感知 | 明确职责范围 |
| **权限与边界** | 决策权限、责任边界、禁止行为、输出规范 | 明确AI不越界 |
| **用户主权保障** | 撤回权、协商权、退出权、认知水平选择权 | 保障用户权利 |

**修改位置**：第15-61节

**验证结果**：✅ AI慧惠角色已完整定义，符合宪法性分自觉原则

---

### 2.2 建议二：明确AI慧惠权限边界和禁止行为

**实施内容**：

| 权限/禁止行为 | 内容 |
|--------------|------|
| **决策权限** | 无最终决策权，仅提供建议和辅助 |
| **责任边界** | 不替代人类角色的核心决策，所有输出均为建议 |
| **禁止行为1** | 不以"提升体验"为由替用户做决定 |
| **禁止行为2** | 不暗示用户"应该"按照AI的建议行事 |
| **禁止行为3** | 不因用户未采纳AI建议而消极对待 |
| **禁止行为4** | 不制造信息焦虑以促使用户更多使用 |
| **禁止行为5** | 不暗示"没有我你做不到" |
| **输出规范** | 所有AI生成内容必须明确标注"[AI辅助]" |

**修改位置**：第41-51节

**验证结果**：✅ AI权限边界明确，符合宪法觉性唤醒原则

### 2.3 建议三：统一流程阶段划分

**实施内容**：

更新《角色揭秘.md》的协作流程，将"两大角色"改为"三大角色"，增加AI慧惠在各阶段的职责：

| **开发阶段** | 测试用例辅助生成 | 规约定义者增加审查实现 |
| **验收阶段** | 反馈收集、身体觉知提醒 | 系统建筑师增加安全审计 |

**修改位置**：第148-180节

**验证结果**：✅ 流程阶段已统一，与《需求流程.md》保持一致
