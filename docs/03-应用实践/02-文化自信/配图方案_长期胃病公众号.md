# 《长期胃病久治不愈？你需要的不是更多药，而是一把"觉知之刃"》配图方案

> 适用平台：微信公众号  
> 文章总长：约3500字  
> 建议配图数：封面1张 + 正文8张（六幕各1 + 总结1 + 练习区图标点缀）  
> 视觉定调：极简东方美学（水墨+留白）× 科学图解（干净线条）


## 一、统一视觉规范

| 要素 | 规范 |
|------|------|
| **主色调** | 米白/暖灰底 + 石墨黑 + 暖金（光） + 暗红（病兆警示） |
| **辅助色** | 淡蓝（思维能量）、暗金（经络/循环）、绿色（重建） |
| **风格** | 极简东方美学 × 科学图解——水墨质感、留白呼吸、线条克制 |
| **构图原则** | 每图留白 ≥ 40%，不塞满，"让文章呼吸" |
| **图中文字** | 如需出现文字，用宋体/思源宋体，纤细，字号克制 |
| **意象暗线** | 全篇"光与暗的对话"——前四幕暗渐浓，第五幕光切入，第六幕光胜出 |
| **AI生图提示词公式** | `minimalist ink wash style, zen aesthetic, [画面描述], negative space, elegant, no text, muted tones` |


## 二、逐幕配图方案

---

### 🖼️ 封面图

| 项目 | 内容 |
|------|------|
| **画面** | 一把极简线条勾勒的"刀刃"（或一道光的裂隙），切入一团水墨晕染的混沌色块。刃是透明的/发光的，不是金属质感 |
| **色调** | 深灰底 + 一抹暖金色（光刃）+ 标题白色字 |
| **构图** | 居中垂直，刀刃从上向下切开画面，下方微微透出光 |
| **意象** | 觉知之刃切开病态振动闭环 |
| **AI提示词参考** | `minimalist ink wash style, zen aesthetic, a transparent glowing blade of light cutting through dark ink clouds, vertical composition, negative space, elegant, muted dark tones with single warm gold accent` |

## 四、执行建议

- **生图工具**：Midjourney / DALL·E / 国产（通义万相、文心一格）均可，提示词已适配英文AI工具，国产工具可参考「画面」描述自行翻译
- **风格统一**：建议全部图片用同一模型/同一风格参数批量生成，确保视觉一致性
- **暗色图（封面、四幕、五幕）**：注意公众号暗色模式下白色文字的可读性，建议图片上部留出足够的暗色区域放标题
- **文件规格**：公众号封面 900×383px（2.35:1），正文配图建议宽度 1080px，高度按构图自由，JPEG 压缩至 300KB 以内
- **备选方案**：如AI生图效果不理想，可委托插画师用「水墨+极简线条」风格统一手绘，本方案描述足以作为 Brief 使用
