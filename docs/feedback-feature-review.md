## 一、功能概述

### 1.1 功能定位

**反馈建议功能**是旋量-太极模型网站的核心用户交互模块，旨在收集用户对网站内容、功能和体验的反馈意见，包括纠错、建议、疑问和赞赏四类反馈。该功能通过前端表单收集反馈，经Cloudflare Worker处理后，将反馈内容提交至GitHub Issues进行统一管理。

### 1.2 功能价值

| 价值维度 | 描述 |
|----------|------|
| **用户参与** | 为用户提供便捷的反馈渠道，增强用户参与感 |
| **质量提升** | 通过用户反馈持续改进网站内容和功能 |
| **问题追踪** | 利用GitHub Issues实现反馈的系统化管理 |
| **社区建设** | 促进用户与开发团队的互动交流 |

### 1.3 功能范围

- ✅ 反馈弹窗的打开与关闭
- ✅ 反馈内容填写与验证
- ✅ 反馈类型选择（建议/纠错/疑问/赞赏）
- ✅ 联系方式填写（可选）
- ✅ 图片上传与预览（最多3张）
- ✅ 反馈提交与状态提示
- ✅ 提交成功/失败处理

## 三、实现方案

### 3.1 技术选型

| 层次 | 技术 | 说明 |
|------|------|------|
| 前端 | HTML5 + CSS3 + JavaScript | 原生实现，无框架依赖 |
| 样式 | feedback.css | 独立样式文件 |
| 交互 | feedback.js | 独立脚本文件 |
| 后端 | Cloudflare Workers | Serverless函数，处理API请求 |
| 存储 | GitHub | Issues存储反馈内容，仓库存储图片 |
| 构建 | build.py | Python脚本，复制文件到dist目录 |

### 3.2 文件结构

```
spinor-taiji-model/
├── assets/
│   ├── feedback.css       # 反馈功能样式
│   └── feedback.js        # 反馈功能脚本
├── articles/
│   └── zh/
│       ├── spinor-taiji-model-statement.html   # 声明页面（引用反馈功能）
│       └── spinor-taiji-glossary.html          # 词汇表页面（引用反馈功能）
├── functions/
│   └── api/
│       └── feedback.js    # Cloudflare Worker API
└── scripts/
    └── build.py           # 构建脚本
```

### 3.3 核心实现思路

#### 3.3.1 前端实现

1. **浮动按钮**：固定定位在页面右下角，滚动超过300px时显示
2. **弹窗组件**：遮罩层 + 模态框结构，支持多种关闭方式
3. **表单验证**：实时验证内容长度，显示字数统计和错误提示
4. **文件处理**：支持点击选择和拖拽上传，FileReader实现预览
5. **提交逻辑**：收集表单数据和文件Base64编码，发送POST请求

#### 3.3.2 服务端实现

1. **请求解析**：支持JSON和FormData两种格式
2. **参数验证**：白名单校验类型和来源，长度限制
3. **文件安全**：扩展名白名单过滤，Base64大小限制
4. **图片上传**：调用GitHub API上传文件到仓库
5. **Issue创建**：生成标题和正文，添加标签，调用GitHub Issues API

## 四、界面设计

### 4.1 设计规范

| 元素 | 设计规范 |
|------|----------|
| **主色调** | 橙色渐变 (#f39c12 → #e67e22) |
| **按钮样式** | 圆角24px，阴影效果 |
| **弹窗尺寸** | 最大宽度440px，占屏幕90% |
| **响应式** | 移动端自动适配 |
| **字体** | PingFang SC, Microsoft YaHei |

### 4.2 组件设计

#### 4.2.1 浮动按钮

```css
.feedback-float {
    position: fixed; bottom: 90px; right: 30px;
    background: linear-gradient(135deg, #f39c12, #e67e22);
    color: white; border: none; border-radius: 24px;
    padding: 12px 18px; box-shadow: 0 4px 15px rgba(243, 156, 18, 0.4);
}
```

**显示逻辑**：
- 默认隐藏（opacity: 0, visibility: hidden）
- 滚动超过300px时显示（添加.show类）
- 滚动事件触发updateFloatButtons函数
